SmartOrder for Restaurants

Smart order is a java based restaurant order system which I have built in order to practice my object oriented programming and design patterns. 
The system I created allows users to create food orders, customize burgers, calculate totals, process payments and to print recipts.

Features:
- Create food items using facotry pattern
- Build custom burgers using the builder pattern
- Store multiple items in an order
- Calculate order totals
- Process payments using different payment types
- Manage all orders using Singleton
- Print the customer receipts
- Organize code into clean packages


How to run:
1. Clone the repo
2. Open the project in a ide
3. run Main.java
